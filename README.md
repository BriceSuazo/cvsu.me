# CvSU.me - a Social Media for Cavite State University

## Stack

- [T3 Turbo](https://github.com/t3-oss/create-t3-turbo)

- [tRPC](https://trpc.io/)
- [Clerk Auth](https://clerk.dev/)
- [Drizzle ORM](https://github.com/drizzle-team/drizzle-orm)
- [PlanetScale serverless JS driver](https://github.com/planetscale/database-js)
